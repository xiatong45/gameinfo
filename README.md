# 我的世界信息显示GameInfo模组二改版

## 功能介绍

GameInfo模组可以在游戏中显示当前的游戏信息，包括：

- 游戏信息展示(FPS、天数、时间、坐标、生物群系、版本、现实时间)
- 装备信息展示
